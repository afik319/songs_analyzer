הוראות הפעלה:
1. לחצו פעמיים על הקובץ venv.bat והמתינו לסיום פעולתו. שלב זה הוא חד פעמי. אין צורך לבצע אותו בכל פעם.
2. התקינו docker desktop.
3. פתחו את הטרמינל:

	במערכת הפעלה windows עליכם להריץ את הפקודה הבאה:
	docker run -d --name mssql-songs-db -p 1433:1433 -v songs_volume:/var/opt/mssql afik319/mssql-songs-db
 
	במערכת הפעלה macOS עליכם להריץ את הפקודה הבאה:
	רר
	
	**הערה: אם ברצונכם להשתמש במספר port אחר במקום 1433 (השמאלי), ניתן להחליפו, אך עליכם יהיה גם לעדכן את הקובץ env. לפורט החדש.

4. בכדי להפעיל את המערכת, לחצו פעמיים על הקובץ songs.bat.
